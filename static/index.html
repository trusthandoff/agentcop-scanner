secure AI agents

From static code analysis to runtime behavioral monitoring, AgentCop gives your team complete visibility into agent security posture.

Prompt Injection Detection

Identifies unsanitized user inputs that could manipulate your agent's system prompt or override safety instructions.

Taint Flow Analysis

AST-based data flow tracking follows tainted sources (user input, env vars, HTTP) through your code to dangerous sinks (eval, exec, subprocess).

AgentCop is a free open source AI agent security scanner. It detects prompt injection (LLM01), insecure output handling (LLM02), supply chain vulnerabilities (LLM05), plugin security issues (LLM07), and excessive agency (LLM08) using real AST taint analysis. It supports LangGraph, CrewAI, AutoGen, LlamaIndex, Haystack, Semantic Kernel, OpenClaw, and Moltbook. pip install agentcop. Free. MIT licensed. No signup required. agentcop.live
